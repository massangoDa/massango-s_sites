    実行方法

    Githubではgit cloneのやり方を紹介しています。

    https://github.com/massangoDa/massango-s_sites

    このプロジェクトはNode.jsを使用していますので、以下の手順に従って実行してください。

    Node.jsをインストールしていない場合

    まず、https://nodejs.org/download/にアクセスして、Node.jsをインストールしてください。

    Node.jsをインストールしたら

    まず、このプロジェクトをダウンロードしてください。(Githubのcloneでもok)

    次に、コマンドプロンプトを開いて、このプロジェクトのディレクトリに移動してください。

    コマンドプロンプトから


                    
node index.js

    で実行することができます

    httpのポート80で建てる設定になっていますので、ご自身の環境に合うようにindex.jsを変更してください。



    貢献

    このプロジェクトは、以下のソフトやライブラリなどをを使わせていただきました。

    • Visual Studio Code: プログラム作成
    • Node.js: JavaScriptの実行環境
    • Express: Webアプリケーションフレームワークとしての基盤を提供 Contributors

    • http: HTTPサーバーを作成 Maintainers
    • fs: ファイルシステムを操作 Maintainers
